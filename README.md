## IO (Input/Output) и Stream API в Java 
 
В этом модуле мы сфокусируемся на работе с вводом и выводом данных в Java, а также на использовании Stream API. IO (Input/Output) позволяет вам взаимодействовать с внешними источниками данных, а Stream API предоставляет удобный способ обрабатывать коллекции данных. 
 
## Что вы изучите 
 
В этом модуле вы изучите следующие темы: 
 
- Чтение и запись данных из и в файлы 
- Работа с различными типами потоков для обработки данных 
- Сериализация и десериализация объектов 
- Работа с сетевым вводом/выводом 
- Использование Stream API для функциональной обработки коллекций данных
